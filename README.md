## How to run the app

To build the project use **'pnpm build'**

To start the project use **'pnpm start'**

To test the project use **'pnpm test'**

## Folders structure

### `dist/`

- **Description:** Generated by the build process, contains the distribution files
- **Files:** Compiled and minified files ready for deployment

### `node_modules/`

- **Description:** Contains installed packages
- **Files:** Libraries and modules

### `src/`

- **Description:** Contains the source code of the app
- **Files:** JavaScript, CSS, HTML index files

### `src/calculator/`

- **Description:** Calculator logic
- **Files:** JavaScript files and tests for this files
